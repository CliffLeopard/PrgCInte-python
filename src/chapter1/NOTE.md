# Chapter1:集体智慧导言

### 集体智慧

* 为了创造新的想法，而将一群人的行为，偏好或者思想组合在一起

### 机器学习

* 将一组**数据**传递给**算法**
* **算法**推断出这些数据的属性相关的**信息**
* 借助信息，算法**预测**出未来可能会出现的其他数据

### 机器学习局限性

* 受限于在大量模式之上的归纳能力
* 对新数据的决策
* 过度归纳

### Example

* Google:PageRank算法
* Amazon/Netflix:推荐商品，音乐
* Hollywood Stock Exchange:市场预测
* eHarmony:交友匹配
